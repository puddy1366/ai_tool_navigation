## 🎯 产品概述

### 产品定位
一个专注于 AI 工具收集与分享的个人导航网站，旨在帮助用户快速发现和使用优质 AI 工具。

### 核心价值
- **精选**：严格筛选优质 AI 工具，确保质量
- **分类**：清晰的分类体系，便于快速查找
- **简洁**：简洁优雅的界面设计，提升用户体验
- **动态**：炫酷的视觉效果，增强科技感

### 目标用户
- AI 技术爱好者和从业者
- 需要使用 AI 工具的创作者
- 希望提升工作效率的职场人士
- 对 AI 技术感兴趣的学习者

## 🎬 动画效果

### 全局背景效果
- 科技宇宙背景图片（fixed 定位）
- 深色遮罩（85% 不透明度）
- 粒子动画
- 动态渐变背景

### 自定义动画

#### 渐变色流动
```css
@keyframes gradient-shift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

#### 光晕脉冲
```css
@keyframes pulse {
  0%, 100% {
    transform: translateX(-50%) scale(1);
    opacity: 0.8;
  }
  50% {
    transform: translateX(-50%) scale(1.2);
    opacity: 0.4;
  }
}
```

#### 卡片交互
- 悬停：上移 4px + 边框发光
- 过渡时长：300ms
- 缓动函数：ease-out

#### 工具列表交互
- 悬停：背景色变化 + 右移 4px + 箭头显示
- 过渡时长：200ms

## 📱 响应式设计

### 断点定义
- **移动端**：< 640px
- **平板**：640px - 1024px
- **桌面端**：≥ 1024px

### 响应式策略

#### 主页
- **移动端**（< 640px）
  - Hero 标题：36px
  - 分类卡片：单列布局
  - 侧边栏：垂直堆叠
  - Footer：居中对齐
- **平板**（640px - 1024px）
  - 分类卡片：两列布局
  - 侧边栏：底部显示
- **桌面端**（≥ 1024px）
  - 分类卡片：三列布局
  - 侧边栏：右侧固定

#### 其他页面
- **移动端**：最大宽度 100%，内容区垂直堆叠
